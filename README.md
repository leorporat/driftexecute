# DriftExecute Monorepo

Start everything with one command from this folder.

## One-time setup

```bash
npm install
```

## Run all services

```bash
npm run dev:all
```

This starts:

- Frontend (Next.js) on `http://localhost:3000`
- Node backend on `http://localhost:3001`
- ML API (FastAPI) on `http://127.0.0.1:8001`
